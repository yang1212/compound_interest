```
let tree = {
  val: 1,
  left: {
    val: 2,
    left: {
      val: 4,
      left: null,
      right: null
    },
    right: {
      val: 5,
      left: null,
      right: null
    },
  },
  right: {
    val: 3,
    left: {
      val: 6,
      left: null,
      right: null
    },
    right: {
      val: 7,
      left: null,
      right: null
    }
  }
}
```
**解题思路**
1) 将问题划分为独立子问题

2) 将子树翻转左右子树即可

<br/>

### 实现

```javaScript
function revertTree(node) {
  if (!node) { return null }
  return {
   val: node.val,
   left: revertTree(node.right),
   right: revertTree(node.left)
  }
}

```
