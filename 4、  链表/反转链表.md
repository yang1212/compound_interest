给你单链表的头节点 head ，请你反转链表，并返回反转后的链表。
 

示例 1：
```
输入：head = [1,2,3,4,5]
输出：[5,4,3,2,1]
```
<br/>

**解题思路**
1) 定义两个指针： front 和 last 
2) 每次让 front 的 next 指向 last
3) 反转完成之后，front 和 last 同时往前移动一个位置
4) 循环上述过程，直至 front 到达链表尾部，得到last则为所求反转的链表


### 实现

```javaScript
function revertList(head) {
  let last = null
  front = head
  while(front) {
    let temp = front.next
    front.next = last
    last = front
    front = temp
  }
  return last
}
```
